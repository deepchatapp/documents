## 系统要求

DeepChat 支持以下操作系统:

- Windows 10/11 (64位)
- macOS 10.15 或更高版本
- Ubuntu 20.04 或更高版本 (以及其他主流Linux发行版)

建议的最低硬件配置:

- 处理器: 双核处理器，2.0GHz或更高
- 内存: 4GB RAM (8GB或更高更佳)
- 磁盘空间: 500MB可用空间
