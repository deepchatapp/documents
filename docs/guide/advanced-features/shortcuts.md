# 快捷键配置

快捷键是提高 DeepChat 使用效率的重要工具，可以让您无需使用鼠标就能快速完成各种操作。本页面将介绍如何查看、使用快捷键。

## 快捷键概览

DeepChat 提供了全面的快捷键支持：

- **内置快捷键**：预设的标准快捷键覆盖常用操作
- **多平台支持**：为 Windows、macOS 和 Linux 提供适配的快捷键

## 默认快捷键

### 基本操作

| 操作     | Windows/Linux | macOS   |
|--------|---------------|---------|
| 新建对话   | `Ctrl+N`      | `Cmd+N` |
| 放大字体   | `Ctrl++`      | `Cmd++` |
| 缩小字体   | `Ctrl+-`      | `Cmd+-` |
| 重置字体   | `Ctrl+0`      | `Cmd+0` |
| 打开设置   | `Ctrl+,`      | `Cmd+,` |
| 打开设置   | `Ctrl+,`      | `Cmd+,` |
| 清除聊天历史 | `Ctrl+L`      | `Cmd+L` |
| 隐藏窗口   | `Ctrl+W`      | `Cmd+W` |
| 关闭软件   | `Ctrl+Q`      | `Cmd+Q` |
