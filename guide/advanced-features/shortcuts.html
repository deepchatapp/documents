DeepChat

快捷键配置

快捷键是提高 DeepChat 使用效率的重要工具，可以让您无需使用鼠标就能快速完成各种操作。本页面将介绍如何查看、使用和自定义快捷键。

快捷键概览

DeepChat 提供了全面的快捷键支持：

  • 内置快捷键：预设的标准快捷键覆盖常用操作
  • 自定义快捷键：可根据个人习惯修改或添加快捷键
  • 快捷键组：针对不同功能区域的快捷键集合
  • 快捷键搜索：快速查找特定操作的快捷键
  • 多平台支持：为 Windows、macOS 和 Linux 提供适配的快捷键

快捷键界面

查看和管理快捷键的方式：

访问快捷键设置

  1. 从设置页面：

    • 打开设置页面（Ctrl+, 或 Cmd+,）
    • 选择"快捷键"分类

  2. 快捷键列表视图：

    • 列表显示所有可用的快捷键
    • 按功能类别分组
    • 支持搜索特定命令

  3. 快捷键参考卡片：

    • 按下 F1 键打开帮助面板
    • 选择"快捷键参考"选项
    • 快速查看常用快捷键列表

这里应放置一张快捷键设置界面的截图，显示快捷键列表和自定义选项。

自定义快捷键

修改快捷键

自定义快捷键的步骤：

  1. 打开快捷键设置:

    • 设置 → 快捷键
    • 找到要修改的命令

  2. 重新绑定快捷键:

    • 点击当前快捷键旁边的编辑图标
    • 按下您想要使用的新快捷键组合
    • 系统会检查冲突并提供冲突解决选项

  3. 移除快捷键:

    • 点击快捷键旁边的"×"图标
    • 确认移除操作

  4. 恢复默认设置:

    • 使用右上角的"恢复默认"按钮
    • 选择恢复所有快捷键或只恢复选定类别

创建自定义快捷键

为未分配快捷键的功能添加快捷键：

  1. 找到未分配的命令:

    • 使用搜索框查找命令
    • 或浏览类别查找未分配快捷键的功能

  2. 添加新快捷键:

    • 点击"未分配"旁边的"+"按钮
    • 按下您想要绑定的按键组合
    • 确认添加

  3. 创建组合命令（高级）:

    • 设置 → 高级 → 自定义命令
    • 创建自定义宏或命令序列
    • 为新创建的命令分配快捷键

快捷键模式

DeepChat 支持多种快捷键模式，适应不同用户的使用习惯：

预设模式

  1. 默认模式:

    • DeepChat 的标准快捷键设置
    • 平衡易用性和功能性

  2. Vim 模式:

    • 为 Vim 用户设计的键位
    • 包括模式切换和 Vim 风格的导航

  3. Emacs 模式:

    • 为 Emacs 用户设计的键位
    • 支持 Emacs 式的命令键和组合键

  4. IDE 模式:

    • 类似于流行 IDE 的键位
    • 适合习惯代码编辑器的用户

自定义模式

创建和管理个人快捷键模式：

  1. 创建模式:

    • 在快捷键设置中选择"创建新模式"
    • 命名新模式
    • 从现有模式复制或从头开始设置

  2. 导出/导入模式:

    • 将自定义模式导出为 JSON 文件
    • 导入他人分享的模式设置
    • 合并不同模式的优点

  3. 模式切换:

    • 使用快捷键 Ctrl+K Ctrl+M（Windows/Linux）或 Cmd+K Cmd+M（macOS）打开模式选择器
    • 快速切换不同的快捷键模式

高级快捷键功能

多步骤快捷键

使用组合按键执行多步操作：

  1. 按键序列:

    • 支持双击键或连续按键
    • 例如 Ctrl+K 然后 Ctrl+S 执行保存所有对话

  2. 快捷键前缀:

    • 使用前缀键区分不同功能组
    • 例如 Ctrl+K 前缀用于文件操作，Ctrl+G 前缀用于导航操作

上下文感知快捷键

根据当前状态提供不同快捷键功能：

  1. 界面上下文:

    • 在不同界面区域提供专用快捷键
    • 例如，输入框、对话历史和侧边栏各有不同的快捷键

  2. 模式上下文:

    • 在不同模式下使用不同快捷键集
    • 例如，在代码块内部有专用的代码编辑快捷键

快捷键宏

记录和重放键盘操作序列：

  1. 宏录制:

    • 开始录制: Ctrl+K Ctrl+R（Windows/Linux）或 Cmd+K Cmd+R（macOS）
    • 执行一系列操作
    • 停止录制: 再次按下录制快捷键

  2. 宏播放:

    • 播放最后录制的宏: Ctrl+K Ctrl+P（Windows/Linux）或 Cmd+K Cmd+P（macOS）
    • 保存命名的宏并通过命令面板访问

触摸栏和手势支持

针对现代设备的附加控制方式：

MacBook Touch Bar 支持

为配备 Touch Bar 的 MacBook 提供的快捷功能：

  1. 上下文操作:

    • 显示当前上下文的相关操作
    • 快速访问常用功能

  2. 自定义触摸栏:

    • 在设置中自定义 Touch Bar 按钮
    • 添加常用操作到 Touch Bar

触控板手势

支持多指手势控制：

  1. 导航手势:

    • 两指滑动: 滚动历史记录
    • 三指滑动: 在对话之间切换
    • 捏合/展开: 缩放内容

  2. 自定义手势:

    • 在设置中定义自定义手势
    • 将复杂操作映射到简单手势

无障碍支持

为各种用户提供的快捷键辅助功能：

  1. 屏幕阅读器兼容性:

    • 所有快捷键操作都有屏幕阅读器提示
    • 使用 Ctrl+Alt+/（Windows/Linux）或 Cmd+Option+/（macOS）查看当前上下文的可用操作

  2. 替代快捷键:

    • 为需要单手操作的用户提供替代键位
    • 支持粘滞键和多功能键

常见问题解答

快捷键相关问题

问题解决方案
快捷键无效检查是否有其他应用占用了相同的快捷键；确认功能在当前上下文中可用；尝试重启应用
无法修改特定快捷键一些核心快捷键可能无法更改；尝试使用不同的按键组合；检查是否有按键冲突
快捷键设置未保存确保点击"保存"按钮；检查权限问题；尝试导出设置后再导入
找不到特定功能的快捷键使用快捷键设置中的搜索功能；查阅完整文档；确认功能是否支持快捷键
多设备快捷键不一致使用设置同步功能；手动导出/导入快捷键配置；确保设备间版本一致

最佳实践

高效使用快捷键

提高快捷键使用效率的建议：

  1. 循序渐进学习:

    • 从最常用的快捷键开始掌握
    • 每周学习和使用几个新快捷键
    • 使用键盘快捷键小抄（可在帮助菜单中打印）

  2. 根据工作流定制:

    • 为您最常执行的操作设置易记的快捷键
    • 考虑键位的人体工程学，减少手指拉伸
    • 在多个应用程序间保持一致的快捷键习惯

  3. 利用快捷键提示:

    • 开启"快捷键提示"功能（设置 → 界面 → 显示快捷键提示）
    • 使用命令面板（Ctrl+P 或 Cmd+P）搜索并查看操作的快捷键

  4. 按功能组学习:

    • 一次专注学习一类相关操作的快捷键
    • 例如，先掌握所有导航快捷键，再学习编辑快捷键
