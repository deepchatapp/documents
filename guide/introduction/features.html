DeepChat 特性详解

多模型支持 核心功能

DeepChat兼容多种大型语言模型，包括但不限于：

OpenAI GPT系列
支持GPT-3.5-turbo、GPT-4等模型，提供流畅自然的对话体验。
本地模型
可接入各种开源大语言模型，适合对隐私有要求的场景。
自定义API
支持接入自定义API服务，满足各种特殊需求。

不同的模型有各自的优势和特点，用户可以根据实际需求选择合适的模型。

文档处理能力 高级功能

DeepChat支持上传多种格式的文档作为附件，并与大模型进行问答：

📄
支持格式：PDF、Word、TXT、Markdown等
📚
长文档处理：能够处理长篇文档，并提取关键信息
🔄
上下文保持：在对话过程中保持对文档内容的理解

Artifacts实时交互 独特功能

直接在聊天界面预览和交互生成的内容：

  • 代码预览：实时语法高亮和格式化
  • 图表展示：支持多种图表类型的生成和展示
  • 游戏交互：支持简单的游戏互动体验
  • 表格数据：结构化展示表格数据

MCP功能 高级配置

Model Control Panel (MCP) 提供灵活的模型控制能力：

  • 系统提示词：定制模型的行为和响应风格
  • 温度调节：控制输出的创造性和多样性
  • 输出格式控制：指定输出的结构和格式
  • 上下文窗口调整：控制模型记忆的对话长度

多平台支持 兼容性

DeepChat支持多种操作系统平台：

  • Windows系统：支持Windows 10/11
  • macOS：支持Intel和Apple Silicon芯片
  • Linux：支持主流Linux发行版

知识库集成 开发中

即将推出的知识库功能将提供：

  • 知识库管理：创建、编辑和管理专属知识库
  • 智能检索：基于语义的智能检索能力
  • 实时更新：知识库内容的实时更新
  • 自定义权重：对不同知识源设置不同的权重
